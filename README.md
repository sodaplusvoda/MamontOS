# MamontOS x86_64

![MamontOS Banner](assets/mamontlogo.png)

Это моя учебная 64-битная Операционная Система написанная на Ассемблере NASM и C.

Проект базируется на концепции "недо-микроядра". Цель - оставить в пространстве ядра только критически важные функции управления системой, но оставить драйвера для дисков и файловую систему.
Все остальные сервисы (драйверы периферии, терминал, и возможно даже GUI, если его добавлю) реализуются как изолированные программы, загружаемые с диска.

В системе уже есть загрузчик, GDT, простой обработчик прерываний вместе с PIC (В будущем перееду на APIC) и PMM.

На счет PMM, в MamontOS реализована гибридная трехуровневая система распределения физических страниц по 4 КБ:
1. Early Allocator для начальной загрузки.
2. Bitmap для низкоуровневого учета всех страниц.
3. Buddy Allocator как основной рабочий механизм. Реализован он с помощью Gemini, заранее извиняюсь.

Также вывод используется с помощью прямой записи в текстовый буфер VGA.
